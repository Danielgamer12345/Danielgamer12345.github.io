# 🥐 Villa Moura Pães | E-commerce de Pães Congelados

![Status do Projeto](https://img.shields.io/badge/Status-Em%20Desenvolvimento-orange)
![Licença](https://img.shields.io/badge/License-MIT-green)

A **Villa Moura Pães** combina a tradição da panificação artesanal com a praticidade da tecnologia moderna. Este repositório contém o código-fonte da nossa vitrine virtual focada em pães ultracongelados produzidos diariamente.

---

## 🎯 Objetivo
Proporcionar uma experiência de compra rápida e intuitiva para clientes de atacado (padarias e hotéis) e varejo, destacando a qualidade da nossa fermentação natural.

## ✨ Destaques da Interface
* **🌓 Dark Mode Nativo:** Alternância de cores para acessibilidade e estilo.
* **🔢 Contador de Fornadas:** Script dinâmico que exibe a escala de produção em tempo real.
* **📱 Mobile First:** Site totalmente otimizado para navegação via smartphone.
* **💬 Conversão via WhatsApp:** Integração direta para fechamento de pedidos.

## 🛠️ Tecnologias
* **Frontend:** HTML5, CSS3 (Grid & Flexbox), JavaScript.
* **Design:** Google Fonts (*Playfair Display* & *Poppins*).
* **Ícones:** Emojis e FontAwesome.
